DirectX12を使って現代的なレンダリング技術を学ぶ実験用、勉強用自作３D描画エンジン。

以下進捗。

**Render Graph（レンダーグラフ）で描画を管理。**

-> どの順番で描くと効率が良いか」「メモリをどう節約するか」を賢く管理する仕組み

**2種類の描画モードを切り替えられる**

-> Forward（フォワード）レンダリング： 素直に物体を描画する、基本的な方式。

-> Deferred（ディファード）レンダリング： 色や奥行きの情報を先に保存して、後から光の計算をまとめてやる方式

**3Dモデルを表示して、自由に動き回れる**

-> assimpを使って、FBX等のファイルを読み込み描画。

**設定をいじれるメニュー画面（GUI）**

-> ImGuiを使って、プログラムを動かしながら設定を変えられる
